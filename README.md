# Campus Evaluation System

## Overview

This repository contains the implementation of multiple frontend and backend modules developed as part of a campus evaluation assessment. The project demonstrates proficiency in REST APIs, React, Node.js, middleware development, logging mechanisms, and priority-based notification processing.

## Installation

### Clone the Repository

```bash
git clone <repository-url>
```

### Navigate to the Project Directory

```bash
cd Campus-Evaluation-FE
```

### Install Dependencies

For Frontend:

```bash
cd notification-app-fe
npm install
```

For Backend:

```bash
cd ../notification-app-be
npm install
```

For Vehicle Scheduler:

```bash
cd ../vehicle-scheduler-be
npm install
```

---

## Running the Applications

### Start Frontend

```bash
cd notification-app-fe
npm run dev
```

### Start Backend

```bash
cd notification-app-be
npm start
```

### Start Vehicle Scheduler

```bash
cd vehicle-scheduler-be
node index.js
```
